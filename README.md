# ブロック崩しゲーム集

4つの異なるアプローチで実装されたブロック崩しゲームのコレクションです。ゲーム開発の学習用に、それぞれ異なる特徴と実装方法を持たせています。

## 🎮 ゲーム紹介

### 1. クラシック版
**シンプルで分かりやすい基本プレイ**

初心者でも楽しめる、シンプルで分かりやすい基本的なブロック崩しゲーム。ゲーム開発の基礎を学ぶのに最適です。

**主な特徴:**
- 基本的な衝突判定
- レベルシステム
- スコア機能
- ライフシステム

**学習ポイント:**
- Canvas APIの基本操作
- ゲームループの実装
- 衝突判定の基本
- 状態管理

### 2. パワーアップ版
**戦略的なアイテム収集が鍵！**

アイテムを集めてパワーアップ！パドルが大きくなったり、ボールが増えたりする戦略的なゲーム。

**パワーアップアイテム:**
- 🔴 **パドル拡大**: パドルが大きくなる
- 🔵 **マルチボール**: ボールが増える
- 🟢 **ボール加速**: ボールが速くなる
- 🟡 **スコア2倍**: 一定時間スコアが2倍になる
- ⚪ **ライフ回復**: ライフが1つ回復する

アイテムの効果は時間制限付き。タイミングよく使って、より高いスコアを目指してください！

**学習ポイント:**
- アイテムシステムの実装
- タイマー管理
- 動的なゲーム要素の追加
- 状態の永続化

### 3. 物理エンジン版
**リアルな物理演算が楽しめる！**

リアルな物理演算でブロックが崩れ落ちる！重力と衝突を体験できる高度な実装。

**物理システム:**
- 重力の影響を受けるブロック
- 衝突時の反発効果
- ブロック同士の衝突も発生
- 摩擦による減速

物理演算により、従来とは違った戦略的なプレイが可能です。

**学習ポイント:**
- 物理エンジンの基礎
- 衝突検出アルゴリズム
- 衝突応答の実装
- 物理シミュレーション

### 4. 特殊効果版
**華やかなエフェクトと特殊ブロックで大迫力！**

爆弾ブロック、マルチボール、特殊な動き！最もエキサイティングで複雑なバージョン。

**特殊ブロックの種類:**
- 💣 **爆弾ブロック**: 周囲のブロックも破壊
- ⭐ **スターブロック**: ボールが3つに分裂
- 🌀 **回転ブロック**: ボールの軌道を曲げる
- ⚡ **加速ブロック**: ボールが速くなる
- 🛡️ **硬いブロック**: 2回当たらないと破壊されない

ブロックを破壊すると美しいパーティクルエフェクトが発生。視覚的にも楽しめる、最も派手なバージョンです！

**学習ポイント:**
- 特殊効果システム
- パーティクルシステム
- 視覚効果の実装
- 複雑なゲームロジック

## 🎯 操作方法

### 基本操作（全ゲーム共通）
- **マウス移動**: パドルを動かす
- **矢印キー / A/Dキー**: パドルを左右に動かす
- **スタートボタン**: ゲームを開始
- **一時停止ボタン**: ゲームを一時停止/再開

## 💻 技術情報

このゲームは以下の技術で実装されています：
- **HTML5 Canvas**: ゲーム描画
- **Vanilla JavaScript**: ゲームロジック
- **CSS3**: スタイリング

外部ライブラリは一切使用していません。すべての機能を純粋なJavaScriptで実装しています。
