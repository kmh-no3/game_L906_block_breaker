# ブロック崩しゲーム集

4つの異なるアプローチで実装されたブロック崩しゲームのコレクションです。ゲーム開発の学習用に、それぞれ異なる特徴と実装方法を持たせています。

## ゲーム一覧

### 1. クラシック版 (`game1_classic`)
**難易度: 基本**

シンプルで分かりやすい基本的なブロック崩しゲーム。ゲーム開発の基礎を学ぶのに最適です。

**特徴:**
- 基本的な衝突判定
- レベルシステム
- スコア機能
- ライフシステム

**学習ポイント:**
- Canvas APIの基本操作
- ゲームループの実装
- 衝突判定の基本
- 状態管理

### 2. パワーアップ版 (`game2_powerups`)
**難易度: 中級**

アイテムを集めてパワーアップ！パドルが大きくなったり、ボールが増えたりする戦略的なゲーム。

**特徴:**
- パワーアップアイテム（5種類）
  - 🔴 赤: パドルが大きくなる
  - 🔵 青: ボールが増える
  - 🟢 緑: ボールが速くなる
  - 🟡 黄: スコアが2倍になる
  - ⚪ 白: ライフが回復する
- マルチボール対応
- タイマー機能付きパワーアップ

**学習ポイント:**
- アイテムシステムの実装
- タイマー管理
- 動的なゲーム要素の追加
- 状態の永続化

### 3. 物理エンジン版 (`game3_physics`)
**難易度: 上級**

リアルな物理演算でブロックが崩れ落ちる！重力と衝突を体験できる高度な実装。

**特徴:**
- 物理演算システム
- 重力システム
- 衝突応答（反発係数）
- ブロック同士の衝突
- 摩擦システム

**学習ポイント:**
- 物理エンジンの基礎
- 衝突検出アルゴリズム
- 衝突応答の実装
- 物理シミュレーション

### 4. 特殊効果版 (`game4_special`)
**難易度: 最上級**

爆弾ブロック、マルチボール、特殊な動き！最もエキサイティングで複雑なバージョン。

**特徴:**
- 特殊ブロック（5種類）
  - 💣 爆弾ブロック: 周囲のブロックも破壊
  - ⭐ スターブロック: ボールが3つに分裂
  - 🌀 回転ブロック: ボールの軌道を曲げる
  - ⚡ 加速ブロック: ボールが速くなる
  - 🛡️ 硬いブロック: 2回当たらないと破壊されない
- パーティクル効果
- 爆発システム
- ボールのトレイル効果

**学習ポイント:**
- 特殊効果システム
- パーティクルシステム
- 視覚効果の実装
- 複雑なゲームロジック

## 操作方法

すべてのゲームで共通の操作方法:

- **マウス**: パドルを動かす
- **矢印キー / A/Dキー**: パドルを左右に動かす
- **スタートボタン**: ゲームを開始
- **一時停止ボタン**: ゲームを一時停止/再開

## 技術スタック

- **HTML5 Canvas**: ゲーム描画
- **Vanilla JavaScript**: ゲームロジック
- **CSS3**: スタイリング

外部ライブラリは使用していません。すべての機能を純粋なJavaScriptで実装しています。

## 学習のポイント

### 1. コードの比較
各ゲームの実装を比較することで、同じゲームでも異なるアプローチがあることがわかります。

### 2. 段階的な複雑さ
基本から最上級まで、段階的に複雑になっているので、順番に学習することで理解が深まります。

### 3. 実装パターン
- ゲームループの実装方法
- 衝突判定のアルゴリズム
- 状態管理の方法
- 視覚効果の実装

### 4. 拡張のヒント
各ゲームは独立しているので、自由に拡張や改造が可能です。
